Theme craig

Theme Craig is a futuristic Grocery Odoo E-Commerce theme aimed at next generation entrepreneurs. The theme design is excellent with the use of the mega menu, a grid like a slider, box layout to showcase exciting products and much more. Theme Craig design aims to be simple yet modern with an interesting choice of on-screen elements. Sections for the new product, on sale products, blogs, etc. are provided on the homepage.

Just like any other modern theme by 73Lines, Craig is a completely responsive and retina ready, making it ready to be rendered on any screen size. Another key feature of the theme is the drag and drop based design. You can create interesting layouts, edit all the elements using the drag and drop builder. Customization doesn't end here as the theme also comes with a powerful admin panel. The admin panel enables the webmaster to make the most out of the theme functionality.The theme is compatible with Odoo v10 Community and Odoo v10 Enterprise and many modules can be used with the theme to extend its functionality. Overall, Craig is a great theme for your next project for Odoo Ecommerce.
